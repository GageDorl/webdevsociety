The Softball Statistics project is a tool designed to analyze and improve softball players' batting performance through detailed statistical tracking. It features a user-friendly interface built with JavaScript, HTML, CSS, and Bootstrap, allowing users to view and manage player stats with CRUD operations. The back end, developed with .NET 8, Entity Framework Core, and MySQL, securely handles data storage and management, with Swagger UI assisting in testing. The project aims to implement graphs for tracking player progress over multiple games. Despite challenges like multi-game tracking and CORS issues, it demonstrates innovative problem-solving and effective integration of front-end and back-end technologies to create a valuable resource for softball teams and coaches.
